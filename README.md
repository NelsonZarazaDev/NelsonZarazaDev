# About Me

👋 ¡Hola! Soy **Nelson Mauricio Navarro Zaraza**, un estudiante de Ingeniería de Sistemas y desarrollador Full Stack apasionado por la creación de soluciones tecnológicas innovadoras.

## 🌐 Desarrollo Web
Me dedico a explorar y dominar el mundo del desarrollo web, creando interfaces atractivas y funcionales que mejoran significativamente la experiencia del usuario. Me enfoco en:
- **Frontend:** HTML, CSS, Bootstrap, Tailwind, JavaScript, React
- **Backend:** Spring Boot, Php

## 🔍 Bases de Datos
Estoy en constante aprendizaje y perfeccionamiento en el diseño y gestión de bases de datos para garantizar un rendimiento eficiente y una gestión segura de la información. Mis habilidades incluyen:
- **Bases de datos relacionales:** MySQL, PostgreSQL
- **Bases de datos no relacionales:** MongoDB
